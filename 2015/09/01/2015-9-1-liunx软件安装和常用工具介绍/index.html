Linux软件安装和常用工具介绍

Linux系统软件安装方法yum和rpm

  1. yum和rpm的安装区别
    yum 是网络安装(依赖互联网环境)；从「源」上下载安装包，若存在依赖关系自动下载依赖包并安装。
    rpm 是本地安装；安装包存在本地，存在依赖关系的安装包需自己手动逐一安装；个人修改编译过的安装包只能用rpm安装。

  2. yum和rpm的基本安装过程（Java为例）
    • 「yum命令」安装的基本过程
      1) 查看系统是否已安装java；
      2) 若已安装的java不合期望则将其卸载；
      3) 查看yum库中可安装的java rpm安装包；
      4) 选择适合的java rpm包进行安装；
      5) 检查java版本信息确保安装成功。
      [root@localhost ~]# ping www.baidu.com         # 检查网络是否正常连通
[root@localhost ~]# yum list installed java*   # 查看资源库中已经安装的java rpm包
[root@localhost ~]# yum -y remove java*        #卸载；参数-y：当需要判断输入[Y/N]时默认为Y，无需输入
[root@localhost ~]# yum list java*             # 查看资源库中可以安装的java rpm包
[root@localhost ~]# yum -y install java-1.7.0-openjdk*   # 当显示"Complete!"时即安装完毕
[root@localhost ~]# java -version                        # 查看当前java版本


    • 「rpm命令」安装的基本过程
      1) 查看系统是否已安装java；
      2) 若已安装的java不合期望则将其卸载；
      3) download或copy适合自己的java rpm安装包；
      4) 选择适合的java rpm包进行安装；
      5) 检查java版本信息确保安装成功。
      注：可用“rpm -qa | grep java | xargs rpm -e —nodeps”命令将java相关文件一次性全部卸载，--nodeps忽略依赖关系。
      [root@localhost ~]# rpm -qa | grep java               # 查看是否已安装java
[root@localhost ~]# rpm -e java-1.7.0-xxx...          # 卸载；rpm -e卸载不支持通配符，卸载对象名称必须完整输入
[root@localhost ~]# wget -c http://download.../../../jdk-8u60-linux-x64.rpm  # 使用wget工具在线下载jdk安装包
[root@localhost ~]# smbclient //ip地址/共享文件夹 -U username%password         # 使用samba工具通过共享文件夹拷贝所需jdk安装包
[root@localhost ~]# rpm -ivh jdk-8u60-linux-x64.rpm   # 安装java rpm安装包，-ivh显示安装进度
[root@localhost ~]# java -version                     # 查看当前java版本

linux系统如何下载软件、安装包—wget下载工具

  • wget是linux、UNIX系统上常用命令行下载工具。
    • wget特点：网络上自动下载工具，支持http、https、ftp的tcp/ip协议下载，同时还是支持http代理和断点续传。
    • wget安装：# yum -y install wget
    • wget用法：# wget [options 选项] [url 下载地址]
    • wget示例：# wget -c http://download.oracle.com/otn-pub/java/jdk/8u60-b27/jdk-8u60-linux-x64.rpm　　# 下载java rpm包
    • 博客推荐：「wget实例用法介绍」

linux系统和windows系统之间文件夹共享—samba共享工具

  • sabma是在linux、UNIX系统上实现SMB协议的免费工具软件。

    • samba特点：由服务器和客户端构成，有诸多套件配套使用，提供smb.conf文件可配置。
    • samba安装：# yum -y install samba.*
    • samba套件安装：# yum -y install samba-server* samba-doc* samba-common* samba-client*
    • samba常用套件：samba-server、samba-doc、samba-common、samba-client四类
    • samba套件简介：
      [samba-server]：是Samba的服务器软件包，Windows主机访问Linux主机上Samba服务器共享的资源时需要安装。
      [samba-client]：是Samba的客户端工具，Linux主机访问Windows主机上共享的资源时需要安装。
      [samba-common]：是Samba的相关工具包，必安程序之一。
      [samba-doc]：是Samba的库文件包，必安程序之一。
    • 博客推荐：「CentOS下Samba服务器的安装与配置」

    Q：本地安装虚拟机如何实现windows和虚拟机linux之间的共享呢？
    A：使用Samba-client的smbclient命令访问Windows主机共享文件夹，然后获取资源。
    Step：
    1) 在windows主机新建文件夹设为共享文件夹并赋予权限，将可共享资源复制到该共享文件夹中。
    2) 在linux主机上安装Samba工具及相关套件(samba-client必装)。
    3) 使用smbclient命令访问windows主机共享文件夹，语法：# smbclient //ip/共享文件夹 -U username%password。
    4) 在smb：\>模式下：输入ls查看共享文件夹内容，再get filename复制资源到Linux主机上，最后exit或Ctrl+C退出。
    注：smbclient语法中参数[-U]必须大写，否则报错。
    或者使用虚拟机提供“共享文件夹”配置，实现windows和虚拟主机linux的共享。
    又或者使用 SecureCRT (SSH工具)的 “Connect SFTP Session(Alt+p)” 通过SFTP实现 windows 和虚拟主机 linux 共享。
